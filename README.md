## 🛠️ Technical Stack & Architecture

This project is built using modern software engineering practices to ensure scalability and clean code:

* **State Management:** `GetX` for reactive UI updates and efficient dependency injection.
* **Networking:** `Dio` for robust HTTP requests with custom interceptors and error handling.
* **In-App Browsing:** `webview_flutter` to provide a seamless reading experience without leaving the app.
* **Theming:** Dynamic **Dark/Light Mode** support managed via GetX.
* **API Integration:** Real-time data aggregation from **News API**.

---

## ✨ Key Features
* **Dynamic Categories:** Instantly filter news by Business, Sports, Technology, and more.
* **Search Functionality:** Find specific articles using optimized API queries.
* **Persistence:** Theme preferences are saved for a consistent user experience.
* **Responsive UI:** Optimized for various screen sizes and orientations.
